# Midjourney

Midjourney — это ИИ-сервис для генерации изображений по текстовому описанию (текст → картинка).

## Что это?

Ты описываешь словами, какую картинку хочешь, а Midjourney создаёт её с помощью искусственного интеллекта. Можно создавать фото, иллюстрации, картины в любом стиле.

## Как начать?

1. Зарегистрируйся в Discord (если нет аккаунта)
2. Зайди на сервер Midjourney в Discord
3. Напиши команду `/imagine` и опиши, что хочешь увидеть
4. Через несколько секунд получишь 4 варианта картинки!

Есть бесплатный пробный период, потом нужна подписка.

## Что умеет?

- **Генерация изображений**: создаёт картинки по описанию
- **Разные стили**: фото, иллюстрации, картины, 3D-рендеры
- **Вариации**: можешь создавать варианты понравившейся картинки
- **Улучшение**: можно улучшить выбранную картинку (upscale)
- **Изменения**: можно вносить изменения в существующую картинку

## Уровни сложности

### Уровень 1: Базовое использование
Просто описывай, что хочешь увидеть.

**Примеры**:
- `/imagine a cute cat playing with a ball`
- `/imagine sunset over the ocean`
- `/imagine a futuristic city`

### Уровень 2: Продвинутое использование
Используй параметры и техники для лучших результатов.

**Техники**:
- **Стили**: добавь `--style raw` или `--style 4.0`
- **Соотношение сторон**: `--ar 16:9` для широкого формата
- **Качество**: `--quality 2` для более детальных картинок
- **Семена**: `--seed 12345` для воспроизводимости

**Примеры**:
- `/imagine a magical forest --style 4.0 --ar 16:9`
- `/imagine portrait of a warrior --quality 2`

## Как работает процесс?

1. **Команда**: пишешь `/imagine [описание]`
2. **Генерация**: Midjourney создаёт 4 варианта (занимает ~1 минуту)
3. **Выбор**: выбираешь понравившийся вариант (кнопки U1, U2, U3, U4 для улучшения или V1, V2, V3, V4 для вариаций)
4. **Улучшение**: можно улучшить выбранную картинку или создать варианты

## Полезные техники промптинга

### Описание деталей
Чем подробнее описание, тем лучше результат:
- "a cat" → "a fluffy orange cat playing with a red ball in a sunny garden"

### Стили
Указывай стиль:
- "in the style of Van Gogh"
- "photorealistic"
- "3D render"
- "watercolor painting"

### Параметры
Используй параметры для контроля:
- `--ar 16:9` — соотношение сторон
- `--quality 2` — качество (больше деталей)
- `--stylize 750` — стилизация
- `--v 6` — версия модели

## Ограничения

- Работает только через Discord (нет веб-интерфейса)
- Нужна подписка после пробного периода
- Может создавать не совсем то, что ожидаешь (нужно экспериментировать)
- Есть ограничения на контент (нельзя генерировать запрещённое)

## Полезные советы

1. **Будь конкретным**: детальное описание = лучший результат
2. **Экспериментируй**: пробуй разные формулировки
3. **Используй вариации**: если результат близок, создавай варианты
4. **Изучай примеры**: смотри, что создают другие, и учись
5. **Комбинируй стили**: "photo of X in the style of Y"
