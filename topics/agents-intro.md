# Агенты: что это и зачем

**Агент** — это ИИ-система, которая может не только отвечать на вопросы, но и выполнять действия: искать информацию в интернете, использовать инструменты, планировать задачи и выполнять их пошагово.

## Чем агент отличается от обычного чат-бота?

**Обычный чат-бот**:
- Отвечает на вопросы
- Генерирует текст
- Не может выполнять действия

**Агент**:
- Может использовать инструменты (поиск в интернете, калькулятор, API)
- Может планировать последовательность действий
- Может выполнять сложные многошаговые задачи
- Может принимать решения на основе результатов

## Как работает агент?

1. **Получает задачу**: «Найди лучшие рестораны в Москве и составь план ужина на вечер»
2. **Планирует действия**:
   - Найти рестораны через поиск
   - Отфильтровать по рейтингу и отзывам
   - Составить план с адресами и временем
3. **Выполняет действия**: использует инструменты для поиска информации
4. **Анализирует результаты**: выбирает лучшие варианты
5. **Формирует ответ**: предоставляет готовый план

## Примеры агентов

- **AutoGPT**: агент, который может выполнять сложные задачи автономно
- **LangChain Agents**: фреймворк для создания агентов с различными инструментами
- **Claude с инструментами**: Claude может использовать калькулятор, поиск и другие инструменты
- **Корпоративные агенты**: могут работать с базами данных, отправлять email, создавать отчёты

## Инструменты агентов

Агенты могут использовать:
- **Поиск в интернете**: для актуальной информации
- **Калькуляторы**: для вычислений
- **API различных сервисов**: для получения данных
- **Базы данных**: для работы с информацией
- **Файловые системы**: для чтения и записи файлов
- **Email и мессенджеры**: для коммуникации

## Зачем нужны агенты?

Агенты позволяют автоматизировать сложные многошаговые задачи:
- Исследование темы с поиском информации
- Планирование поездок с бронированием
- Анализ данных из разных источников
- Автоматизация рутинных рабочих процессов

## Ограничения

- Агенты могут делать ошибки в планировании
- Могут зацикливаться на задачах
- Требуют контроля для важных операций
- Могут быть дорогими из-за множества вызовов API
