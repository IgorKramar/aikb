# Шаблоны промптов для разных задач

Шаблоны промптов — это готовые структуры, которые можно адаптировать под свои задачи. Это экономит время и помогает получать более качественные результаты.

# Шаблон 1: Анализ текста

Проанализируй следующий текст:

[ВСТАВЬ ТЕКСТ]

Выполни анализ по следующим пунктам:
1. Основная тема и идея
2. Ключевые аргументы
3. Стиль и тон
4. Целевая аудитория
5. Сильные и слабые стороны

# Шаблон 2: Написание текста

Напиши [ТИП ТЕКСТА] на тему "[ТЕМА]".

Требования:
- Аудитория: [ОПИСАНИЕ АУДИТОРИИ]
- Тон: [ТОН]
- Длина: [КОЛИЧЕСТВО СЛОВ/ПРЕДЛОЖЕНИЙ]
- Стиль: [СТИЛЬ]
- Обязательно включи: [ЧТО ВКЛЮЧИТЬ]

Структура:
1. [ПУНКТ 1]
2. [ПУНКТ 2]
3. [ПУНКТ 3]

# Шаблон 3: Решение проблемы

Помоги решить проблему: [ОПИСАНИЕ ПРОБЛЕМЫ]

Проанализируй ситуацию и предложи решение:
1. Опиши проблему детально
2. Определи возможные причины
3. Предложи 3 варианта решения
4. Оцени плюсы и минусы каждого
5. Рекомендуй лучший вариант с обоснованием

# Шаблон 5: Сравнение

Сравни [ОБЪЕКТ 1] и [ОБЪЕКТ 2] по следующим критериям:
- [КРИТЕРИЙ 1]
- [КРИТЕРИЙ 2]
- [КРИТЕРИЙ 3]

Формат: таблица с колонками "Критерий", "[ОБЪЕКТ 1]", "[ОБЪЕКТ 2]", "Вывод"
В конце дай общий вывод и рекомендацию.

# Шаблон 6: Планирование

Составь план [ТИП ПЛАНА] для [ЦЕЛЬ].

Учти:
- Сроки: [СРОКИ]
- Ресурсы: [РЕСУРСЫ]
- Ограничения: [ОГРАНИЧЕНИЯ]

Структура плана:
1. Цели и задачи
2. Этапы с датами
3. Необходимые ресурсы
4. Возможные риски и решения
5. Критерии успеха

# Шаблон 7: Редактирование текста

Отредактируй следующий текст:

Задачи:
- Исправить грамматические и орфографические ошибки
- Улучшить стиль и читаемость
- Сохранить основной смысл и тон
- Оптимизировать длину (если нужно)

В конце покажи список внесённых изменений.

# Шаблон 8: Генерация идей

Придумай [КОЛИЧЕСТВО] идей для [ЗАДАЧА/ТЕМА].

Требования:
- Идеи должны быть [ХАРАКТЕРИСТИКИ]
- Учти ограничения: [ОГРАНИЧЕНИЯ]
- Для каждой идеи укажи:
  * Краткое описание
  * Преимущества
  * Потенциальные сложности
  * Как реализовать

Формат: нумерованный список с подпунктами

# Как использовать шаблоны?

  1. Выбери подходящий шаблон для своей задачи
  2. Заполни переменные (текст в [СКОБКАХ])
  3. Адаптируй под свои нужды — добавь или убери пункты
  4. Сохрани свой вариант для будущего использования
