# Что такое модель и как она работает

**Модель** — это обученная нейросеть, которая умеет выполнять определённые задачи: отвечать на вопросы, генерировать текст, создавать картинки.

Представь библиотекаря, который прочитал миллионы книг и запомнил, как люди обычно отвечают на вопросы, пишут тексты, описывают картинки. Этот библиотекарь — и есть модель.

## Как работает модель?

1. **Обучение**: модель «читает» миллиарды текстов, картинок, видео из интернета
2. **Запоминание паттернов**: она учится, какие слова обычно идут друг за другом, как выглядят объекты на картинках
3. **Генерация**: когда ты задаёшь вопрос, модель «угадывает», какой ответ будет наиболее вероятным, основываясь на том, что видела раньше

## Разные модели для разных задач

- **Текстовые модели** (ChatGPT, Claude): умеют писать тексты, отвечать на вопросы
- **Модели для картинок** (Midjourney, DALL-E): умеют создавать изображения по описанию
- **Голосовые модели**: умеют распознавать и генерировать речь
- **Специализированные модели**: для перевода, анализа данных, программирования

## Почему модели разные?

Модели обучаются на разных данных и для разных целей:
- ChatGPT обучался на текстах — поэтому хорошо пишет
- Midjourney обучался на картинках — поэтому хорошо рисует
- Специализированные модели обучались на узких задачах — поэтому лучше в своей области

## Важно понимать

Модель не «знает» факты в привычном смысле. Она не хранит базу данных. Вместо этого она умеет генерировать правдоподобные ответы, основываясь на паттернах, которые увидела при обучении.
