# Что такое промпт и зачем он нужен

Промпт (от английского "prompt" — подсказка, команда) — это текст, который ты пишешь ИИ, чтобы получить нужный ответ.

Простыми словами: промпт — это твой вопрос или задание для ИИ.

# Зачем нужны промпты?

ИИ не умеет читать мысли. Если ты напишешь «помоги», он не поймёт, с чем именно помочь. А если напишешь подробный промпт «помоги составить меню на день рождения на 10 человек, бюджет 5000 рублей, есть вегетарианцы», то получишь точный и полезный ответ.

# Из чего состоит хороший промпт?

  1. Задача: что именно нужно сделать
  2. Контекст: дополнительная информация, которая поможет
  3. Формат: как должен выглядеть ответ (список, таблица, текст)
  4. Стиль: тон и уровень сложности

# Примеры промптов

# Простой промпт

«Напиши рецепт борща»

# Хороший промпт

«Напиши рецепт борща для начинающих. Пошаговая инструкция с точными количествами. Время приготовления каждого этапа. Советы для тех, кто готовит впервые»

Видишь разницу? Второй промпт даст гораздо более полезный ответ.

# Типы промптов

  • Вопросы: «Почему небо голубое?»
  • Задания: «Напиши план статьи про...»
  • Исправления: «Исправь ошибки в этом тексте...»
  • Творческие: «Придумай сказку про...»

# Примеры из жизни

  • Промпт для написания письма: «Напиши деловое письмо с просьбой о встрече»
  • Промпт для объяснения: «Объясни квантовую физику простыми словами»
  • Промпт для планирования: «Составь план тренировок на месяц»
