# Практическое руководство по Midjourney

Midjourney — мощный инструмент для создания изображений, но чтобы получить хороший результат, нужно знать, как правильно составлять промпты.

# Основы работы с Midjourney

# Что такое промпт в Midjourney?

Промпт — это описание того, что вы хотите увидеть. Чем подробнее описание, тем лучше результат.

# Структура хорошего промпта

  1. Основное описание: что вы хотите увидеть
  2. Детали: стиль, цвета, композиция
  3. Технические параметры: угол съёмки, формат, качество

# Примеры промптов

# Простой промпт

a cat playing with a ball
1

# Хороший промпт

low angle photo of a person, high fashion pose, friendly smile,
beach setting, eco style, colors: black white blue,
full body view, portrait similarity
1
2
3

# Технические команды

# Угол съёмки

  • low angle photo — фотография с низкого угла
  • high angle — с высокого угла
  • eye level — на уровне глаз

# Поза и стиль

  • high fashion pose — поза в стиле высокой моды
  • casual pose — непринуждённая поза
  • standing — стоя
  • sitting — сидя

# Описание лица

Можно описать детали:

  • Форма лица
  • Цвет глаз
  • Эмоции (friendly smile, serious expression)
  • Причёска
  • Макияж

# Локация

  • beach — пляж
  • city street — городская улица
  • desert — пустыня
  • studio — студия

# Стиль

  • eco style — эко-стиль
  • minimalist — минимализм
  • vintage — винтаж
  • modern — современный

# Процесс создания изображения

# Шаг 1: Создайте базовый промпт

Начните с простого описания:

photo of a person on the beach, friendly smile
1

# Шаг 2: Добавьте детали

Уточните стиль и параметры:

low angle photo, high fashion pose, friendly smile,
beach setting, eco style, colors: black white blue
1
2

# Шаг 3: Тестируйте и корректируйте

  • Если результат не подходит — уточните промпт
  • Уберите лишние детали, если они мешают
  • Добавьте нужные детали, если их не хватает

# Шаг 4: Используйте вариации

Midjourney предлагает 4 варианта. Вы можете:

  • Улучшить выбранный вариант (U1, U2, U3, U4)
  • Создать вариации (V1, V2, V3, V4)

# Лайфхаки

# Лайфхак 1: Используйте ChatGPT для создания промптов

ChatGPT может помочь описать детали:

"Опиши лицо человека: форма лица, цвет глаз, эмоции,
причёска, макияж. Стиль: дружелюбный, современный"
1
2

Затем используйте это описание в Midjourney.

# Лайфхак 2: Начинайте с простого

Не пытайтесь описать всё сразу. Начните с базового промпта, посмотрите результат, затем уточняйте.

# Лайфхак 3: Убирайте лишнее

Если результат не подходит, попробуйте убрать некоторые команды:

  • Убрали low angle photo — результат улучшился
  • Убрали описание эмоций — лицо стало лучше

# Лайфхак 4: Комбинируйте стили

Можно комбинировать разные стили:

high fashion pose, casual clothing, beach setting,
eco style, minimalist colors
1
2

# Работа с лицами

# Добавление лица в изображение

Некоторые приложения (например, FaceSwap) позволяют:

  1. Создать изображение в Midjourney
  2. Заменить лицо на фотографии на нужное лицо
  3. Получить результат с портретным сходством

# Описание лица в промпте

Можно описать лицо детально:

face: oval shape, blue eyes, friendly smile,
brown hair, natural makeup
1
2

Но для точного портретного сходства лучше использовать приложения для замены лиц.

# Параметры Midjourney

# Соотношение сторон

  • --ar 16:9 — широкий формат
  • --ar 1:1 — квадрат
  • --ar 9:16 — вертикальный (для сторис)

# Качество

  • --quality 2 — высокое качество (больше деталей)
  • --quality 1 — стандартное качество

# Стилизация

  • --stylize 750 — высокая стилизация
  • --stylize 250 — низкая стилизация

# Версия модели

  • --v 6 — последняя версия
  • --v 5 — предыдущая версия

# Ограничения

  • Midjourney работает только через Discord
  • Нужна подписка после пробного периода
  • Результат может отличаться от ожиданий
  • Нужно экспериментировать с промптами

# Когда использовать Midjourney?

✅ Хорошо подходит, если:

  • Нужны изображения для соцсетей
  • Создаёте визуальный контент
  • Нужны иллюстрации для статей
  • Хотите экспериментировать с визуальными стилями

❌ Лучше использовать другие инструменты, если:

  • Нужны точные фотографии реальных людей
  • Требуется конкретное портретное сходство
  • Нужны изображения с точными деталями
