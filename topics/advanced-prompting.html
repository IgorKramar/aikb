# Продвинутые техники промптинга

Когда базовых техник промптинга уже недостаточно, можно использовать более продвинутые подходы для получения лучших результатов.

# Техника 1: Few-shot learning (обучение на примерах)

Покажи модели несколько примеров того, что ты хочешь, и она поймёт паттерн.

Пример 1:
Вопрос: Что такое Python?
Ответ: Python — это язык программирования для разработки веб-приложений, анализа данных и автоматизации.

Пример 2:
Вопрос: Что такое JavaScript?
Ответ: JavaScript — это язык программирования для создания интерактивных веб-страниц и приложений.

Пример 3:
Вопрос: Что такое Rust?

# Техника 2: Chain-of-Thought (цепочка рассуждений)

Попроси модель объяснить свои рассуждения шаг за шагом.

❌ Обычный промпт: «Реши задачу: У Маши 5 яблок, она отдала 2. Сколько осталось?» ✅ Chain-of-Thought: «Реши задачу пошагово, объясняя каждый шаг: У Маши 5 яблок, она отдала 2. Сколько осталось?»

Модель ответит:

  1. У Маши было 5 яблок
  2. Она отдала 2 яблока
  3. Осталось: 5 - 2 = 3 яблока

# Техника 3: Role-playing (ролевая игра)

Назначь модели роль эксперта.

❌ «Объясни квантовую физику» ✅ «Ты — учитель физики с 20-летним стажем. Объясни квантовую физику ученику 10 класса, используя простые аналогии и примеры из жизни»

# Техника 4: Итеративное уточнение

Начинай с общего, потом уточняй.

Шаг 1: «Напиши план статьи про искусственный интеллект» Шаг 2: «Для пункта 3 из плана напиши подробный текст на 300 слов» Шаг 3: «Добавь к этому тексту 3 примера из жизни»

# Техника 5: Разделение задач

Разбивай сложные задачи на части.

Вместо: «Напиши полную статью про ИИ с введением, основной частью, выводами, таблицей сравнения и списком источников»

Делай:

  1. «Составь структуру статьи про ИИ»
  2. «Напиши введение для этой статьи»
  3. «Напиши основную часть по пунктам структуры»
  4. «Создай таблицу сравнения разных типов ИИ»
  5. «Напиши выводы и список источников»

# Техника 6: Ограничения и правила

Чётко задавай правила для ответа.

«Напиши email коллеге. Правила:

  • Максимум 100 слов
  • Тон: профессиональный, но дружелюбный
  • Обязательно упомяни дату встречи
  • Закончи вопросом
  • Не используй эмодзи»

# Техника 7: Рефрейминг (переформулировка)

Если ответ не подошёл, попроси переформулировать с другим акцентом.

«Переформулируй этот ответ, сделав акцент на практическом применении, а не на теории»
