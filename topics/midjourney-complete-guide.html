# Midjourney: Полное руководство для начинающих (2025)

Подробное руководство по использованию Midjourney — одного из самых мощных инструментов для генерации изображений с помощью ИИ.

# Регистрация и начало работы

# Шаг 1: Регистрация

  1. Перейдите на Midjourney.com (opens new window)
  2. Войдите через аккаунт Google
  3. Перейдите на веб-сайт для создания изображений

# Шаг 2: Интерфейс

Основные страницы:

  • Исследование — просмотр общедоступных изображений
  • Создание — генерация новых изображений
  • Редактирование — работа с созданными изображениями
  • Персонализация — настройка стиля
  • Организация — управление галереей
  • Чат — общение в сообществе
  • Задания — опросы и активность

# Философия работы с промптами

# Начинайте с малого

Правильный подход:

  1. Начните с простого описания
  2. Постепенно добавляйте детали
  3. Тестируйте каждый элемент отдельно

Пример:

Шаг 1: "коала"
Шаг 2: "коала, одетая как джентльмен"
Шаг 3: "коала, одетая как джентльмен, стоит у входа в бар"
Шаг 4: "коала, одетая как джентльмен, стоит у входа в бар, курит сигарету на улице, винтажный полароид"

# Порядок имеет значение

Важно: порядок упоминания элементов влияет на результат.

  • Тема → Обстановка: "гитарист на тускло освещённой улице" (фокус на гитаристе)
  • Обстановка → Тема: "тускло освещённая улица с гитаристом" (фокус на обстановке)

# Параметры генерации

# Стилизованное значение (Stylize)

Что это? Параметр от 0 до 1000, который определяет баланс между красотой и точностью.

Как работает:

  • 0-100: максимальная точность, но может быть менее красиво
  • 100 (по умолчанию): баланс между красотой и точностью
  • 100-1000: более красивые изображения, но меньше соответствуют промпту

Когда использовать:

  • Низкое значение (0-50): когда важна точность деталей
  • Среднее (100): для большинства задач
  • Высокое (500-1000): когда важнее художественность, чем точность

# Параметр хаоса (Chaos)

Что это? Параметр от 0 до 100, который определяет разнообразие результатов.

Как работает:

  • 0 (по умолчанию): предсказуемые, надёжные результаты
  • 2-5: рекомендуемый диапазон для разнообразия
  • Высокие значения: более необычные, но могут быть бессмысленными

Когда использовать:

  • 0: когда нужен предсказуемый результат
  • 2-5: для экспериментов и поиска интересных вариантов
  • Высокие: для абстрактных и необычных изображений

# Стиль Raw

Что это? Режим, который делает изображения более реалистичными и буквальными.

Особенности:

  • Промпты воспринимаются более буквально
  • Меньше художественной интерпретации
  • Лучше для фотореалистичных изображений

Когда использовать:

  • Фотореалистичные изображения
  • Когда нужна точная передача деталей
  • Для документального стиля

# Работа с вариациями

# Типы вариаций

Сильные вариации (Strong Variations):

  • Заметные изменения в изображении
  • Помогают исправить ошибки
  • Полезны для поиска новых направлений

Тонкие вариации (Subtle Variations):

  • Минимальные изменения
  • Сохраняют основную композицию
  • Для тонкой настройки

# Техника "V-образного вращения"

  1. Сгенерируйте изображение
  2. Нажмите на вариацию (V)
  3. Если результат лучше — повторите
  4. Продолжайте, пока не найдёте идеальный вариант

Преимущества:

  • Постепенное улучшение результата
  • Возможность исправить ошибки
  • Поиск лучшей композиции

# Функция ремиксов (Remix)

# Что это?

Ремиксы позволяют изменять ключевые элементы изображения, сохраняя основную композицию.

Как использовать:

  1. Выберите изображение
  2. Включите режим Remix
  3. Измените промпт (цвет, тему, стиль)
  4. Генерируйте новую версию

Важно: удаляйте параметр chaos при использовании ремиксов для лучших результатов.

Пример:

Исходное: "женщина в красном платье"
Ремикс: "женщина в синем платье" (та же поза, другой цвет)

# Редактор Midjourney

# Базовое редактирование

Как использовать:

  1. Выберите изображение
  2. Нажмите "Edit"
  3. Выберите область для редактирования
  4. Напишите новый промпт для этой области
  5. Генерируйте

Совет: выбирайте область большего размера, чтобы избежать ошибок на границах.

# Внешняя раскраска (Outpainting)

Что это? Расширение изображения за его границы.

Применение:

  • Расширение фона
  • Добавление элементов вокруг основного объекта
  • Создание панорам

Как использовать:

  1. Выберите изображение
  2. Выберите "Expand" или "Outpaint"
  3. Укажите направление расширения
  4. Добавьте промпт для новых элементов

# Редактирование середины

Возможности:

  • Редактирование центра изображения
  • Размещение в новой рамке
  • Изменение композиции

# Уменьшение и размещение

Применение:

  • Уменьшение объекта
  • Размещение в новом контексте
  • Создание коллажей

# Профессиональные трюки

Готовые кадры:

  • Используйте готовые кадры для упрощения работы
  • Стирайте внешнюю сторону оригинального изображения для плавного перехода

Изменение промпта:

  • Изменение промпта в редакторе позволяет избавиться от нежелательных элементов
  • Пример: изменение на "стиль ар-нуво" удаляет нежелательный портрет

# Продвинутый редактор

# Доступ

Требования:

  • Более 10 снимков
  • Подписка на 12 месяцев
  • Или годовая подписка

# Возможности

Редактирование по URL:

  • Загрузите изображение по ссылке
  • Или загрузите с компьютера
  • Редактируйте как обычное изображение

Ретекстурирование:

  • Изменение стиля изображения
  • Сохранение формы и композиции
  • Пример: изменение на "аниме 1990-х с яркими цветами"

# Организация галереи

# Настройка просмотра

Параметры:

  • Размер миниатюр
  • Макет (сетка, список, полный)
  • Рекомендация: полный макет для удобства

# Фильтры

Типы фильтров:

  • По высоким показателям
  • По размеру
  • По версии Midjourney
  • По опубликованным материалам

# Управление изображениями

Действия:

  • "Нравится" — сохранение в избранное
  • "Скрыть" — скрытие из основной галереи
  • Поиск скрытых изображений на организованной странице

# Папки

Создание папок:

  1. Выберите изображения (перетаскивание курсора или Shift+клик)
  2. Создайте папку
  3. Переместите изображения

Умные папки:

  • Автоматически заполняются изображениями по ключевым словам
  • Пример: папка "ночь" с ключевыми словами "ночь" и "средневековье"
  • Удобно для автоматической организации

# Скрытый режим

# Настройки приватности

По умолчанию: все изображения общедоступны

Скрытый режим:

  • Доступен только по тарифному плану Pro ($48-60/месяц)
  • Изображения не видны в общедоступной галерее
  • Полезно для коммерческих проектов

Применение:

# Работа с изображениями в промптах

# Добавление изображений

Как использовать:

  1. Добавьте изображение в промпт
  2. Добавьте текстовое описание
  3. Генерируйте

Пример:

[изображение] + "автомобиль"
1

# Вес изображения

Параметр: от 0 до 3

Как работает:

  • 0: изображение игнорируется
  • 1 (по умолчанию): нормальное влияние
  • 2-3: сильное влияние изображения

Рекомендации:

  • Начните с 1
  • Увеличьте до 2-3, если нужно больше влияния
  • Используйте несколько изображений для комбинирования

# Ссылки на стиль

Что это? Использование изображения как референса стиля, а не содержания.

Параметр: вес стиля от 0 до 1100

Как работает:

  • 0-100: слабое влияние стиля
  • 100-500: среднее влияние
  • 500-1100: сильное влияние стиля

Пример:

[изображение зонтика] + "рисованный стиль" + style weight 500
1

# Описание персонажа (Character Reference)

# Что это?

Функция для создания консистентных персонажей в разных сценах.

Ограничения:

  • Лучше работает с изображениями из Midjourney
  • С января 2025 поддерживает только изображения из Midjourney
  • Не работает с реальными фотографиями

# Как использовать

  1. Добавьте изображение персонажа
  2. Выберите "Character Reference"
  3. Добавьте промпт для новой сцены
  4. Генерируйте

# Вес персонажа

Параметр: от 0 до 100 (по умолчанию 100)

Как работает:

  • 100: использует все данные о персонаже (лицо, одежда, причёска)
  • 50: баланс между лицом и другими элементами
  • 0: фокус только на лице

# Комбинированные подсказки

# Тройной вариант

Комбинация:

  1. Промпт к изображению (содержание)
  2. Ссылка на стиль (художественный стиль)
  3. Ссылка на персонажа (консистентный персонаж)

Пример:

[изображение кошки] + [стиль мультфильма] + [персонаж]
= уникальные фотографии кошки в мультяшном стиле

# Персонализация

# Два варианта

1. Стандартный профиль:

  • Мини-игра с выбором между изображениями
  • Нужно набрать 40 баллов для разблокировки
  • Менее предпочтительный вариант

2. Доска настроений (Mood Board):

  • Собираете коллекцию изображений
  • Создаёте определённый стиль
  • Более предпочтительный вариант

# Создание доски настроений

Источники изображений:

  • С компьютера
  • Ссылки из интернета
  • Из галереи Midjourney

Процесс:

  1. Создайте доску настроений
  2. Добавьте 10-20 изображений в нужном стиле
  3. Сохраните
  4. Используйте при генерации

Пример: доска настроений на основе "сине-жёлтый зонт" создаст стиль в этих цветах

# Стилизованное значение для доски

Параметр: от 100 (по умолчанию) до 1000

Как работает:

  • 100: умеренное влияние доски
  • 500-1000: сильное влияние, более смелые результаты

# Лоскутное шитьё (Quilting)

# Что это?

Экспериментальная функция для создания историй и миров.

Возможности:

  • Создание мира с сюжетными точками
  • Персонажи, настройки, предметы
  • Объединение больших языковых моделей для написания истории

Применение:

  • Создание визуальных историй
  • Разработка концептов для игр/фильмов
  • Создание связанных изображений

# Практические советы

# Для начинающих

  1. Начинайте просто: не пытайтесь описать всё сразу
  2. Экспериментируйте: пробуйте разные параметры
  3. Изучайте других: смотрите промпты популярных изображений
  4. Сохраняйте: используйте "нравится" для сохранения лучших работ

# Для продвинутых

  1. Используйте перестановки: тестируйте отдельные слова
  2. Комбинируйте техники: изображения + стиль + персонаж
  3. Создавайте доски настроений: для консистентного стиля
  4. Используйте редактор: для точной настройки результатов

# Частые ошибки

❌ Слишком длинный промпт: модель теряет фокус ✅ Оптимально 150-300 слов

❌ Игнорирование параметров: используйте chaos, stylize ✅ Экспериментируйте с параметрами

❌ Ожидание идеального результата с первого раза ✅ Используйте вариации и ремиксы
